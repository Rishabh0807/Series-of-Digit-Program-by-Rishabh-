# Digit Series Program

This Python program creates a continuous number series like 123456789101112 and lets the user find the digit at any specific position.

## Example
Enter the number: 20  
Enter the digit till you want the series to be printed: 30  
The required seq is: 123456789101112131415161718192  
Enter the digit position: 30  
Number at digit 30 is: 2
